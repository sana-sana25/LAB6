# PizzaRecipes - LAB6

--- 

## Description

PizzaRecipes est une application Android développée en Java permettant d'afficher une liste de pizzas avec leurs informations (nom, prix, durée, image) et d’accéder à une fiche détaillée pour chaque recette.

## Objectifs du TP 

Ce projet permet de comprendre :

- L’architecture d’un projet Android (packages : classes, dao, service, adapter, ui)
- L’utilisation d’une ListView avec Adapter personnalisé
- Le passage de données entre activités (Intent)
- La création d’un Splash Screen
- La manipulation d’un modèle orienté objet

## Fonctionnalités
- Affichage d'une liste de pizzas
- Image + nom + prix + durée
- Détails complets (ingrédients, description, étapes)
- Splash Screen au démarrage
- Bouton de partage (ou fermeture de l'application)

## Technologies utilisées
- Java
- Android SDK
- XML (Layouts)
- ListView & Adapter personnalisé

--- 

## Lancement de l'application
- Cloner le projet 
- Ouvrir avec Android Studio
- Lancer sur un émulateur ou téléphone
